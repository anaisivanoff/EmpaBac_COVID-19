Comment avons-nous obtenu les données ?

Nous avons utilisé un script de web scraping pour extraire les données de data.gouv.fr.

Les données concernent les taux de réussite du Bac avant et après le Covid.

Nous avons utilisé les technologies suivantes :

  • Python & BeautifulSoup : pour extraire les données.
  • Pandas : pour structurer les données.
  • Chart.js : pour visualiser les statistiques.

Exemple de code de scraping

import requests
from bs4 import BeautifulSoup

url = "https://www.data.gouv.fr"
response = requests.get(url)
soup = BeautifulSoup(response.text, "html.parser")

# Extraction des données (exemple)
titles = soup.find_all("h2")
for title in titles:
    print(title.text)
