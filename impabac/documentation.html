Documentation du Projet

Comment avons-nous obtenu les données ?

Nous avons utilisé un script de web scraping pour extraire les données des sources suivantes :

  • Résultats Bac 2018 - Ministère de l'Éducation
  • Résultats Bac 2019 - Groupe Réussite
  • Résultats Bac 2020 - Studyrama
  • Résultats Bac 2021 - Studyrama
  • Résultats Bac 2022 - Studyrama
  • Résultats Bac 2023 - Studyrama
  • Résultats Bac 2024 - L'Étudiant

Les données concernent les taux de réussite du Bac avant et après le Covid.

Nous avons utilisé les technologies suivantes :

  • Puppeteer : pour automatiser la récupération des données.
  • Chart.js : pour visualiser les statistiques.
  • Html/Css : pour la partie visuel.
